## PSCAD/EMTDC简介

PSCAD/EMTDC是一款高级的电磁暂态仿真软件，被全球电力行业广泛应用。它结合了直观的图形用户界面(PSCAD)和强大的后台仿真引擎(EMTDC)，允许用户设计、仿真和分析复杂的电力系统行为，特别是在处理包含直流分量在内的瞬变现象方面表现出色。
